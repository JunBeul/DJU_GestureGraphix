# GestureGraphix web

## 프로젝트 설명
GestureGraphix의 접근을 위한 웹페이지 입니다.

로그인, 회원가입, ID/PW 찾기 페이지가 있으며

가상공간이라는 컨셉을 생각하여 계속 변화하는 공간을 표현하였습니다.

BG는 그라데이션이 지속적으로 변화하여 유동적인 느낌을 주었고

UI는 글라스모피즘을 사용하여 가상 현실에 떠있는 화면이라는 느낌을 주었습니다.
